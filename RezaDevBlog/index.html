10/24/16 - ArcGIS Javascript API - Creating a custom form for feature creation

I've aways felt the default edit feature class / create feature class tool in ArcGIS Javascript API feels a bit too 'GIS nerdy' for the casual user. I'm not a big fan of creating the feature first and then populating the attribute table, and well the attribute table that appears to me just looks plain ugly. In this demo i'll show you how to create a custom form and to save it so it will be ready to be applied to a feature class or a graphic if needed. More... »
